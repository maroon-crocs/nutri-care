# Branded Diet Plan Table PDF

## Goal
Keep the weekly diet plan PDF in table format while making it more polished and clinic-ready.

## Requirements
- Use the existing NutriGuide leaf branding in the PDF header.
- Keep the weekly table with day rows and meal-slot columns.
- Use light colors to improve readability without making the PDF heavy.
- Include patient summary details above the table.
- Keep final instructions and daily notes available in the PDF.
- Preserve direct jsPDF generation; do not use browser print.

## Acceptance
- Normal weekly plans render as a single landscape page when meal text is concise.
- Longer plans can continue across pages cleanly with repeated table headers.
